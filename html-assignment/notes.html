These are five tags that i've learned in html in the last few classes:

The <img> Tag:
  • This tag helps us add images into our document.
  • It's a singular tag.
  • We can add images from the internet or images that we have in our computers.
The <hr> Tag:
  • This tag helps us add a horizontal line to divide our document.
  • It's a singular tag.
The <nav> Tag:
  • This tag helps us add a navigation to our page.
  • It's a paired tag.
  • It can help us navigate to different pages in our page.
The <blockquote> Tag:
  • This tag helps us add a large quote into our document.
  • It's a paired tag.
  • When we use this tag, text gets indented.
The <strong> Tag:
  • This tag helps us make text of our choice become bold.
  • It's a paired tag.
  • When a screen reader it's being used, it will emphasize more text inside this tag.
